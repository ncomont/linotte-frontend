Télécharger le fichier type Il s’agit du fichier vierge au format demandé par l’Etat

Comment compléter le fichier ? Voir le document d’aide officiel pour remplir les champs exigés.

Remplissez les champs obligatoires et les champs facultatifs souhaités. Afin de lier vos données à l’identifiant Taxref, renseignez à minima le champ nom scientifique ou nom vernaculaire. À partir de ce champ, l’outil fera la liaison avec les informations taxonomiques contenues dans Taxref puis transformera vos données au format demandé par l’Etat.

Ce formulaire va vous permettre de créer une nouvelle tâche de liaison automatique de taxons. Vous devez indiquer un nom pour la tâche planifier.
